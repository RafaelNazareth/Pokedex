# Pokédex

Uma Pokédex interativa e moderna que consome a [PokeAPI](https://pokeapi.co/) para exibir informações sobre todos os Pokémon (até a 9ª geração). Desenvolvida com HTML, CSS e JavaScript puro, oferece uma experiência fluida e responsiva para fãs da franquia.

## 🚀 Demonstração

Acesse a demonstração online: https://rafaelnazareth.github.io/Pokedex/

## ✨ Funcionalidades

- **Lista completa** de todos os Pokémon (1025 espécies)
- **Busca em tempo real** por nome ou número
- **Filtro por tipo** (fogo, água, grama, etc.)
- **Modal de detalhes** com informações completas:
  - Estatísticas base (HP, ataque, defesa, etc.)
  - Altura e peso
  - Habilidades
  - Tipos com cores distintas
- **Design responsivo** para todos os dispositivos
- **Carregamento progressivo** com barra de progresso
- **Cache local** para melhor performance
- **Animações suaves** e feedback visual

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização, animações e design responsivo
- **JavaScript (ES6+)** - Lógica e interatividade
- **PokeAPI** - Fonte de dados dos Pokémon
- **Font Awesome** - Ícones
- **Google Fonts (Lato)** - Tipografia

## 📋 Pré-requisitos

Nenhum! O projeto roda diretamente no navegador. Para desenvolvimento local, apenas um servidor HTTP simples é necessário.
