## 💼 Sobre o Projeto

Esta Pokédex interativa é um projeto full-stack front-end que demonstra minhas habilidades em **consumo de APIs REST**, **manipulação dinâmica do DOM** e **design responsivo**. O projeto lista todos os Pokémon até a 9ª geração (1025 espécies) com informações detalhadas e interface intuitiva.

### 🎯 Objetivos Técnicos Alcançados
- ✅ Consumo eficiente da **PokeAPI** com tratamento de requisições
- ✅ **JavaScript assíncrono** com Promises e Async/Await
- ✅ **Manipulação avançada do DOM** para renderização dinâmica
- ✅ **Sistema de cache local** para otimização de performance
- ✅ **Filtros em tempo real** sem recarregamento da página
- ✅ **Design responsivo** com abordagem mobile-first
